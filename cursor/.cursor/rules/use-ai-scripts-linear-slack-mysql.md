Sempre que um comando envolver Linear, Slack ou MySQL, use prioritariamente os scripts locais em `~/ai-scripts`.

Diretrizes:
- Não executar comandos diretos dessas ferramentas se existir script equivalente em `~/ai-scripts`.
- Primeiro verificar scripts disponíveis em `~/ai-scripts` e escolher o mais apropriado.
- Em caso de dúvida entre comando direto e script, preferir o script.
